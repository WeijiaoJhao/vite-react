# vite-react
 - router：react-router-dom
 - store：zustand

## Environment prepare
NodeJS v16.14.2+

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### locally preview production build
```
npm run preview
```
